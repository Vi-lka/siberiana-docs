## API vs Sandbox

Наш API полностью открытый для чтения данных. Вы можете получить любую интересующую Вас информацию для исследования, создания приложений или API.

Для работы с данными вы можете использовать наш API:
  1. Напрямую с любым языком программирования; 
  2. Воспользоваться ПО по типу [Postman](https://www.postman.com/);
  3. Воспользоваться [Sandbox](https://api.siberiana.online/);

Первых два варианта больше подойдут специалистам, имеющим опыт работы с **GraphQL**, языками программирования и спец. ПО.

Для [Sandbox](https://api.siberiana.online/) не требуется профильных навыков, и изучения документации должно хватить для полноценной работы с ней.
